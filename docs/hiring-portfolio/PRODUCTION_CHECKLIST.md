## âœ¨ Post-Production Phase

### Visual Polish
- [ ] Color grade for consistency
- [ ] Adjust brightness and contrast
- [ ] Enhance colors (subtle saturation)
- [ ] Apply color LUT or preset (optional)
- [ ] Ensure brand colors are accurate
- [ ] Add subtle vignette (optional)
- [ ] Review on different monitors/devices

### Text & Graphics
- [ ] Add lower thirds for key points
- [ ] Create call-out text for emphasis
- [ ] Add section markers/titles (optional)
- [ ] Style call-to-action card
- [ ] Add contact information overlays
- [ ] Ensure all text is readable at 1080p
- [ ] Check for typos in all text

### Subtitles & Captions
- [ ] Auto-generate subtitles in editor OR
- [ ] Use subtitle service (Rev, Happy Scribe, Descript)
- [ ] Review auto-generated text for errors
- [ ] Correct all mistakes
- [ ] Break lines at natural pauses
- [ ] Style subtitles (white text, black bg, centered)
- [ ] Export SRT file
- [ ] Create burned-in subtitle version (for platforms without caption support)

### Audio Mastering
- [ ] Normalize voiceover levels (-3 to -6dB peak)
- [ ] Apply light compression to voiceover
- [ ] EQ voiceover (boost clarity, reduce mud)
- [ ] Remove mouth clicks and breaths (if distracting)
- [ ] Balance music levels with voice
- [ ] Ensure consistent volume throughout
- [ ] Test on laptop speakers, headphones, phone

### Final Render
- [ ] **Full version** (3-5 min):
  - Resolution: 1920x1080
  - Frame rate: 24 or 30fps
  - Format: MP4 (H.264)
  - Bitrate: 10-15 Mbps VBR
  - Audio: AAC, 48kHz, 24-bit
  - File size: <500MB
- [ ] **High-quality archive** (for future edits):
  - Highest quality settings
  - Store safely with project files
- [ ] Watch full render start to finish
- [ ] Check for compression artifacts
- [ ] Verify audio/video sync
- [ ] Test playback on multiple devices

### Alternative Versions
- [ ] **Short version** (60-90 sec):
  - Hook + key message + CTA
  - Optimized for LinkedIn/social
- [ ] **Teaser** (30 sec):
  - Hook + intrigue + "watch full story"
  - For ads and email campaigns
- [ ] **Square format** (1:1, 1080x1080):
  - For Instagram/LinkedIn feed
  - Reframe key elements
- [ ] **Vertical format** (9:16, 1080x1920):
  - For Stories, TikTok, Reels
  - Reframe and adjust pacing

## ðŸ”„ Iteration Plan

Based on feedback and data:

### If Attention Drops Early
- [ ] Strengthen hook (first 10 seconds)
- [ ] Move key insight earlier
- [ ] Create shorter version

### If Business Impact Unclear
- [ ] Add more specific metrics
- [ ] Emphasize cost/time savings
- [ ] Include comparison chart

- [ ] Make CTA more specific
- [ ] Lower barrier (shorter call, email first)
- [ ] Add social proof or testimonials

- [ ] Create series for other projects
- [ ] Expand into longer case study
- [ ] Create written companion piece
- [ ] Speak about process at meetups/conferences
